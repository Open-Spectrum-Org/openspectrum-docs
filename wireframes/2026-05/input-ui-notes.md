Here are the three screens. A few design decisions worth calling out:

Screen 1 — Quick Capture is the home the parent lives on. The color-coded tag grid uses four distinct semantic colors (purple = behavior, teal = food, amber = medication, pink = emotion) so categories are instantly scannable even in a stressed moment. No confirmation dialogs — every tap is fire-and-forget.

Screen 2 — Incident Logging opens when the parent taps "Meltdown" from Screen 1 or navigates there. The coral header sets a visual alarm without being aggressive. Severity is a 3-button toggle (not a slider) because fine-grained sliders are frustrating under stress. The "Save + remind" button is key for follow-up observations.

Screen 3 — Daily Dashboard closes the loop — it gives the parent a sense of accomplishment for logging. The 4-stat row at the top is the most scannable summary possible. The timeline uses dot color to encode valence (green = positive, amber = neutral, coral/red = negative) so the day reads at a glance. Insights are phrased as plain observations, not AI output.
