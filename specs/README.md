## User Stories (MVP)

Three core stories that define the MVP:

1. **Log fast**: As a parent, I can log an event (food, medication, behavior) in under 30 seconds
2. **See history**: As a parent, I can view my child's event history as a timeline
3. **Spot patterns**: As a parent, I can see a simple weekly chart of behavior vs medication timing

## Feature Prioritization

See [MVP-ideas.md](./MVP-ideas.md) for the full breakdown of what's in MVP vs later phases.

**MVP summary:**
- Child profile
- Event logging (Food, Medication, Behavior) via text and voice
- Timeline view
- Basic charts
- Local-only SQLite storage
- CSV export

**Explicitly not in MVP:**
- AI recommendations
- Cohort comparison
- Multi-user / caregiver sync
- Plugin ecosystem
- Cloud storage
